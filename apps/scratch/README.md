# Scratch for Maotuwo

This is the `scratch` application within the Maotuwo ecosystem, built upon the official `scratch-editor` mono-repo.

## Setup

1. `npm install` inside the project.
2. `npm start` to run the development server.

## Dev Environment
- **Port:** `8011`
- **Spec / Checkpoints:** See `specs/` directory for tracked development progress.

## Ecosystem
- Managed under the broader Maotuwo framework.
- See `maotuwo-app-spec` in the root workspace project.

## About the Source Repositories
The original scratch-editor README has been renamed to `README-scratch-editor.md` to preserve the Maotuwo context while retaining upstream information.
